머신러닝이 무엇인지, 머신러닝의 대표적인 분석 방법 3가지 (선형 회귀, 로지스틱 회귀, 소프트맥스 회귀)

## 1. 머신러닝

사진과 같이 특이한 고양이 사진을 컴퓨터가 정확한 규칙으로 찾아내는 것은 쉽지 않다.

![이미지](/assets/img/2026-07-16-LLM공부(3)/image(3).png){: width="500" }

머신러닝은 데이터와 결과를 이용해서 규칙을 찾아내는 방법이다. 이렇게 찾아낸 규칙으로 데이터만 가지고 결과값을 유추할 수 있는 프로그램을 만들 수 있다.

![이미지](/assets/img/2026-07-16-LLM공부(3)/image.png){: width="500" }

## 2. 머신러닝 배경지식

### 1. 데이터 분리

- 훈련(Training) → 모델 학습용
- 검증(Validation) → 하이퍼파라미터 튜닝 / 성능 조정용
- 테스트(Test) → 최종 성능 평가용

![이미지](/assets/img/2026-07-16-LLM공부(3)/image(1).png){: width="300" }

### 2. 파라미터 종류

- 하이퍼파라미터 → 사용자가 직접 정하는 값 (학습률, 은닉층 수, 배치 크기 등)
- 파라미터 → 모델이 데이터로부터 학습해 스스로 결정하는 값 (가중치 W, 편향 b)

### 3. 분류와 회귀

- 분류(Classification) → 정해진 카테고리 중 하나를 예측 (이산값)
  - ㄴ 이진 분류 : 2개 중 1
  - ㄴ 다중 클래스 분류 : 3개 이상 중 1
- 회귀(Regression) → 연속적인 실수값을 예측

### 4. 지도, 비지도 학습

- 지도학습 → 레이블(정답)이 있는 데이터로 학습
- 비지도학습 → 레이블 없이 데이터의 구조 / 패턴을 스스로 발견
- 자기지도학습 → 레이블 없는 데이터에서 스스로 레이블을 만들어 학습

### 5. 샘플과 특성

- 샘플(Sample) → 하나의 데이터
- 특성(Feature) → 하나의 독립변수

### 6. 혼동행렬 (Confusion Matrix)

![이미지](/assets/img/2026-07-16-LLM공부(3)/image(2).png){: width="300" }

- 정밀도(Precision) = TP / (TP + FP) → 참이라 예측한 것 중 실제 참 비율
- 재현율(Recall) = TP / (TP + FN) → 실제 참인 것 중 참이라 예측한 비율
- 정확도(Accuracy) = (TP + TN) / 전체

### 7. 과적합과 과소적합

- 과적합(Overfitting) → 너무 과하게 학습한 상태
- 과소적합(Underfitting) → 학습이 부족한 상태 (성능이 더 좋아질 수 있음)

![이미지](/assets/img/2026-07-16-LLM공부(3)/image(4).png){: width="400" }

## 3. 선형 회귀 (Linear Regression)

연속적인 값을 예측할 때 사용

### 1. 가설 세우기

단순 선형 회귀 분석(Simple Linear Regression Analysis)

다중 선형 회귀 분석(Multiple Linear Regression Analysis)

초기의 가중치의 값은 일반적으로는 랜덤하게 설정됨 (도메인 지식 기반으로 설정하는 경우도 있음)

### 2. 비용함수

MSE(Mean Squared Error, 평균 제곱 오차) 사용

### 3. 옵티마이저

경사하강법(Gradient Descent) 사용
