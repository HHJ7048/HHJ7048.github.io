# 자격요건, 우대요건 읽어보기
LLM과 ML을 보안과 융합한 직무는 정확히 정해진 분야가 없다. 하지만 보안SW개발, 보안관제, 보안 컨설팅 등등 다양한 직무에서 활용되고 있고 우대 요건으로도 기입 되어있다.

## 🏢SK쉴더스 SOAR 운영/관리 직무
### 직무
1. SOAR자동화 플랫폼 운영 및 관리
2. 자동화 시나리오 구성 및 관리
3. 신규 모듈 개발 및 유지보수
### 조건
1. Python 라이브러리 활용 가능자
2. 보안솔루션 운영 경험자
3. 원활한 Communication Skill 보유자
### 🔍분석
SOAR 운영관리 : Security Orchestration, Automation and Response의 약자로 반복적으로 도움을 주는 플랫폼을 운영관리 하는 것을 말한다.
여기서는 LLM이나 ML보다 데이터 처리쪽이랑 python을 이용한 개발을 중요하게 보는 것을 확인할 수 있다.
그리고 실제로 보안관련 제품을 운영하거나 개발 해본적 있는 경력자를 우대하는 것을 확인할 수 있다.
해외 법인을 포함한 대규모 인프라를 상대로 보안 솔루션을 제공하기에 커뮤니케이션 능력이 중요하고 다양한 분야의 도메인 지식도 필요해 보인다.
여기서는 ML보다는 보안 솔루션 플랫폼 개발쪽에 더 가깝기 때문에 내가 가고싶은 방향과는 연관성은 있지만 어느정도 거리가 있다.

## 🏢안랩 ML/MLOps 엔지니어
### 직무
1. AI/ML 모델 설계, 개발, 최적화
2. 위협 탐지, 분석, 대응 등을 위한 AI/ML 데이터 구성 및 모델 구축
3. AI/ML 모델의 성능 모니터링 및 지속적인 개선
4. 보안 연구 및 기술 트렌드 분석, 최신 AI/ML 기술 도입
5. 기술 문서 작성 및 보안 팀과 협업을 통한 모델 개발/개선
### 조건
1. 데이터 처리 및 분석, 머신러닝/딥러닝 모델 개발 및 운영 경험
2. Python, Scala 등의 프로그래밍 언어에 능숙하며, 보안 데이터에 대한 이해가 있는 분
3. PyTorch 등의 AI/ML 라이브러리 및 프레임워크 경험
4. Spark 등의 대규모 데이터 처리 경험
5. 악성 코드 분석, 침입 탐지 시스템 등의 보안 관련 지식
6. MLOps 관련 경험
7. 관련 연구 경험
### 🔍분석
앞에 있는 SOAR운영/관리 직무와는 달리 LLM과 ML DL에 더 직접적인 업무이다.
AI/ML모델을 설계 개발하고 최적화 하는 역량을 키워나가야한다. 또한 안랩 기업의 특성에 맞게 악성 코드 분석, 침입 탐지 시스템 등의 보안 관련 지식또한 공부해야한다.
현재 내가 가고싶어하는 방향과 가장 맞는 분야이고 AI/ML과 융합할 수 있는 보안 분야에 대한 정보를 알 수 있었다.

Spark : 용량 데이터를 여러 대의 컴퓨터에 나눠서 빠르게 처리하는 분산 처리 엔진

MLOps는 요즘 새로 생기고 있는 직무로 ML + Operate의 합성어이다. 머신러닝을 개발 운영하는 직무이다.

## 🏢지니언스 보안솔루션 엔지니어 인턴십
### 직무
1. 지니언스 엔지니어 실무자를 도와 인턴십 진행 (서버/네트워크 보안 실무 보조)
### 조건
1. 서버, 네트워크 보안 지식 및 기술 보유자
2. Windows/Linux Server 지식 및 기술 보유자
3. IT 관련 교육 이수자
4. 보안 및 IT 인프라에 대한 관심 (정보보안, 네트워크, AI, OS 관련 동아리/학회 활동, 해킹방어대회 참여 경험, 취약점 분석 관심)
5. 오픈소스 보안도구 및 AI 활용, Python·API·LLM(OpenAI 등)을 활용한 데이터 처리 및 자동화 유경험자
6. 최신 사이버 보안 동향(제로트러스트, 랜섬웨어 등) 관심 및 기술 블로그·Notion·GitHub 기록 습관
7. OS/Network 관련 자격증, 클라우드 서비스 기술, 각종 보안·IT 자격증 보유자
### 🔍분석
인턴십 모집 공고로 진입장벽이 가장 낮다. 그래서 보안 및 IT 인프라에 대한 관심, 기술 블로그 등등 역량보다 관심과 노력에 초점을 맞춘 것을 확인할 수 있다.

여기서 보안솔루션은 LLM ML과는 거리가 있다. 하지만 우대 조건을 확인하면 오픈소스 보안도구 및 AI를 활용하고 Python API LLM을 활용할 줄 아는 경험자를 위주로 모집을 하고있다.
추가적으로 리눅스, 윈도우와 같이 OS지식이나 네트워크 관련된 자격증을 요구하고 있다.

# 채용공고를 바탕으로 직무 구체화하기
안랩에서의 ML/MLOps 엔지니어가 가장 가고싶어하는 방향과 맞았기 때문에 기준으로 잡고 조사했다.

## 🔍어떤 일을 하는가?
ML/MLOps 엔지니어는 ML을 이용해 서비스를 개발, 최적화 하고 운영하는 직무이다. 
또한 안랩의 기업 특성상 보안문제를 해결하고 실시간 공격 탐지 등과 같은 보안 SW 개발쪽으로 초점이 맞춰진다.

1. 데이터 수집 및 전처리 : 네트워크 트래픽, 시스템 로그, 프로그램 등등에서 위협등을 구분해 내기 위해 데이터를 수집하고 전처리한다.
2. 위협 탐지 모델 개발 : 전처리된 데이터를 가지고 ML모델을 학습시켜 맬웨어 탐지, 수상한 네트워크 트래픽 탐지 등등 다양한 모델을 개발한다. (ML을 이용한 보안 SW 개발)
3. 모델을 운영하고 지속적으로 개선시킨다. (MLOps의 주된 직무)

## 🔍어떤 역량이 필요한가?
내가 생각하기에 가장 중요한 역량에 대해서 순위를 매겨보았다.
1. Python 활용능력 : 기본적으로 ML LLM 모델은 Python 라이브러리를 사용한다.
2. ML/DL 모델에 대한 이해
3. 보안 도메인 지식 : 악성코드 분석, IDS, 네트워크 트래픽, 로그 데이터 분석
4. 대규모 데이터 처리 : 관련 연구나 프로젝트로 직접 데이터 처리를 해보았는지
5. 논문 분석, 어학 역량 : 계속 발전해나가는 분야라 꾸준히 공부해나가는 습관도 중요하다.

## 📌앞으로의 계획은 어떻게 되는가? -> 면접, 서류에서 어떻게 보여줄 것인가
1. NLP/ML/DL에 대한 공부 -> 공부내용 블로그에 정리, 연구실 활동
2. 기본적인 CS 지식(네트워크, 컴퓨터 구조, Python 고급문법, OS) -> 수업 + 자격증으로 증명(기사 자격증, 네트워크 관리사, 리눅스 마스터 등등)
3. 보안 도메인 지식 공부 -> 블로그 정리, 보안 동아리 활동
4. 꾸준한 논문 읽기, 영어 공부 -> 논문 세미나 발표자료 모아두기, 어학 공부는 구체화 필요함...
