# 활성화 함수 / (손실함수) / 옵티마이저

## 활성화 함수
비선형의 형태를 띄고있다. 선형 함수를 중복해서 사용해도 결국 선형적으로 표현 가능하기 때문이다. + 목적에 따라서 선형적 층이 사용되기도 한다. ex) Projection Layer라고 하며 대표적으로 Embedding Layer가 있다.

![이미지](/assets/img/2026-07-16-LLM공부(4)/image(4).png){: width="600" }

## 대표적인 활성화 함수의 종류

### Step function(계단함수)

자주 사용되지는 않지만 퍼셉트론의 기본 개념을 잘 설명해주는 함수

![이미지](/assets/img/2026-07-16-LLM공부(4)/계단함수.png){: width="400" }

### Sigmoid function
시그모이드 함수는 출력값을 (0,1)로 조정해서 반환해준다.  

![이미지](/assets/img/2026-07-16-LLM공부(4)/시그모이드함수.png){: width="400" }

단점은 경사하강법을 사용할 때 기울기가 최대 0.25로 중복해서 곱해질 경우 기울기 소실(Vanishing Gradient) 문제가 발생할 수 있다. 
역전파 과정에서 앞단까지 기울기가 잘 전달되지 않게 된다.

![이미지](/assets/img/2026-07-16-LLM공부(4)/image(7).png){: width="400" }

시그모이드 함수는 미분식을 간단한 식으로 표현이 가능하기 때문에 컴퓨터 연산에서 강점을 가진다.

$$ \sigma'(z) = \sigma(z)\left(1 - \sigma(z)\right) $$

### Hyperbolic tangent function
출력값의 범위를 (-1, 1)로 조정해서 반환해준다. 기울기의 최대값은 1로 시그모이드 함수보다 큰 값을 가지고 있어 기울기 소실 문제를 어느정도 보완해준다. 하지만 여전히 기울기 소실문제가 발생한다.  
수식은 tanh()이다.

![이미지](/assets/img/2026-07-16-LLM공부(4)/image(8).png){: width="400" }

### ReLU
인공 신경망 은닉층에서 가장 인기있는 함수로 수식은 max(0, x)이다.  
렐루함수는 음수를 입력하면 0을 출력하고 양수를 입력하면 입력값 그대로를 반환한다. 그리고 양수일경우 기울기가 항상 1이다. 따라서 앞서 설명한 함수들과 달리 은닉층에서 훨씬 더 잘 작동한다. 그리고 연산이 필요한 것이 아니라 단순 임계값을 반환함으로 연산속도도 빠르다.

![이미지](/assets/img/2026-07-16-LLM공부(4)/image(9).png){: width="400" }

하지만 음수면 미분값이 0이 되면서 신경망이 죽을 수 있다. 이 뉴런은 다시 회생하는 것이 매우 어렵다. 이러한 문제를 dying ReLU라고 한다.

### Leaky ReLU
앞에서 설명한 dying ReLU를 보완하기 위해 등장한 ReLU의 변형이다.  
음수일 경우에 아주 작은 미분값을 반환하여 ReLU가 죽는 것을 방지한다.

![이미지](/assets/img/2026-07-16-LLM공부(4)/리키렐루함수.png){: width="400" }

### Softmax function
소프트맥스 함수는 주로 출력층에서 자주 사용된다. 다중 클래스 분류(MultiClass Classification) 문제에서 주로 사용된다. 

![이미지](/assets/img/2026-07-16-LLM공부(4)/소프트맥스함수.png){: width="400" }

## 손실함수(Loss function)
실제값과 예측값의 차이를 수치화 해주는 함수 오차가 클 수록 손실 함수의 값은 커지고 옵티마이저를 통해 최소화하는 매개변수인 가중치 w와 편향 b를 찾는 것을 목표로 한다.

## 손실함수의 종류

### MSE
연속형 변수를 예측할 때 사용 ML부분에서 다뤘으므로 생략

### Cross Entropy
이진 분류와 다중 클래스 분류에서 사용된다. ML에서 다뤘으므로 생략

## 옵티마이저(Optimizer)
손실함수의 값을 최소화 하는 것을 목표로 하며 경사하강법을 사용한다. 경사하강법은 배치 크기와 최소화 방법에 따라 종류가 달라진다.  
배치 크기에 따른 분류는 BGD(Batch Gradient Descent), SGD(Stochastic Gradient Descent), MBGD(Mini-Batch Gradient Descent)가 있다.

## 옵티마이저 종류
### Momentum
모멘텀은 경사하강법에 물리학의 관성을 응용한 방법이다. 접선의 기울기에 한 시점 전의 접선의 기울기 값을 일정한 비율만큼 반영한다. 따라서 Local Minimum에 빠지는 것을 방지할 수 있다.

$$ v_{dW} = \beta_1 v_{dW} + (1-\beta_1)\, dW \qquad \Rightarrow \qquad W = W - \alpha \cdot v_{dW} $$

### Adagrad
각 매개변수에 서로 다른 학습률을 적용시킨다. 변화가 많은 학습률은 작게 설정되고 변화가 적은 학습률은 높게 설정시킨다.
### RMSprop
Adagrad는 반복될 경우 학습률이 지나치게 떨어지게 된다는 단점이 있는데 수식을 대체해서 이를 보완했다.

$$ s_{dW} = \beta_2 s_{dW} + (1-\beta_2)\, dW^{2} \qquad \Rightarrow \qquad W = W - \alpha \cdot \frac{dW}{\sqrt{s_{dW}} + \epsilon} $$

### Adam
Adam은 Momentum과 RMSprop를 합친 방식이다.

$$ W = W - \alpha \cdot \frac{v_{dW}^{corrected}}{\sqrt{s_{dW}^{corrected}} + \epsilon} $$

# 순전파 / 역전파

## 순전파
입력이 각 층의 가중합과 활성화 함수를 거쳐 출력값을 만드는 과정  
입력층 -> 출력층 방향으로 진행 (가중치, 활성화 함수)


## 역전파
각 가중치에 대한 미분값(기울기)를 이용해 오차를 최소화하는 과정  
출력층 ->입력층 방향으로 진행 (가중치 업데이트, 옵티마이저)
